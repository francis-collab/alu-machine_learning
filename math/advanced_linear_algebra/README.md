# Advanced Linear Algebra

Implementation of matrix operations without using NumPy (except for task 5):

- Determinant
- Minor matrix
- Cofactor matrix
- Adjugate matrix
- Inverse matrix
- Matrix definiteness (using NumPy)

Tasks 0–4: pure Python  
Task 5: using numpy

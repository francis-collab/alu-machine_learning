# Plotting with Matplotlib

This project demonstrates fundamental data visualization techniques using **NumPy** and **Matplotlib** (version 3.0) as part of the ALU Machine Learning curriculum.

## Project Overview

The following tasks explore different types of plots commonly used in data analysis:

- Line plots (including cubic functions)
- Scatter plots
- Exponential decay curves with logarithmic scaling
- Multiple overlaid line plots with legends
- Histograms (frequency distribution)
- Multi-plot figure layout
- Stacked bar charts

## Files

| File                    | Description                                          |
|-------------------------|------------------------------------------------------|
| `0-line.py`             | Cubic function plotted as a solid red line           |
| `1-scatter.py`          | Scatter plot of men's height vs weight               |
| `2-change_scale.py`     | Exponential decay of C-14 with logarithmic y-scale   |
| `3-two.py`              | Comparative decay curves of C-14 and Ra-226          |
| `4-frequency.py`        | Histogram of student grades (Project A)              |
| `5-all_in_one.py`       | All previous plots combined in a 3Ã—2 grid layout     |
